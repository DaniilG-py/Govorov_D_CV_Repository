Простой телеграм бот, выполняющий функции магазина.
Реализованы: представление товаров по категориям, добавление в корзину пользователя, очистка корзины пользователя.
Не реализованы функции оплаты товаров, так как бот несет исключительно демонстрационный характер.

Бот реализован на TelegramBotApi, БД - SQLite3, управляется тем же простым SQLite3.

При попытке деплоя на Heroku выявилась следующая проблема:
при запуске бота образуется попытка многопоточного доступа к БД,
что возбуждает исключение о запрете рекурсивного доступа к курсору.

Так как отправка изображений реализована через id этих изображений,
хранящиеся в БД, запуск бота возможен только с оригинальным Телеграм токеном,
так как эти id индивидуальны для каждого бота. 
Для актуализации id избражений требуется запустить специальную утилиту "image_id_updater_util.py",
с Вашим телеграм токеном, которая обновит id изображений в БД.

ИНСТРУКЦИЯ ПО ЗАПУСКУ:

1) Клонировать репозиторий
2) Создать окружение и установить все requirements.txt
3) В image_id_updater_util.py вписать свой телеграм токен в соответствующую переменную и запустить.
4) Запустившемуся боту отправить команду "/run" и дождаться сообщения об окончании обновления изображений
5) Вписать свой телеграм токен в файл config.py
6) Запустить useless_things_shop.py

Если все пункты выполнены привильно, то терминал напишет Вам "Bot has been started".
