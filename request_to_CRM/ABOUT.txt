Задача данного скрипта - создание 50ти клиентов через API известной CRM(Название не разглашается).

По ТЗ 50 клиентов должны иметь:
- Случайные имена,
- Случайные даты рождения (от 01/01/1970 по 31/12/2000),
- Случайные номера телефонов,
- Код клиента по ТЗ имеет следущее условие (
		код у первого из клиентов - равен 1, у второго - 2,
        	у последующих - сумме кодов двух предыдущих клиентов.)

API CRM принимает максимум 2 запроса в секунду.

Все ссылки и данные аутентификации удалены из кода!
